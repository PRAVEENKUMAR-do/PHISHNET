About PhishNet

PhishNet is an ML-based system that classifies any URL as Legitimate or Malicious/Phishing using 20 URL-structural features and a trained Random Forest model. The model achieves 96.74% accuracy on the test set with an ROC-AUC of 98.5%.

Base Paper: Kasar et al. (2025) — Phishing URL Detection using ML. IEEE ICAMIDA. DOI: 10.1109/ICAMIDA64673.2025.11208974
