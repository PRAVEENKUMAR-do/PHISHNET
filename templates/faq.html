Frequently Asked Questions

Everything about PhishNet

What is PhishNet and what does it do?

PhishNet is an ML-based malicious URL detection system that classifies any website link as Legitimate or Phishing/Malicious using 20 URL structural features and a trained Random Forest model achieving 96.74% accuracy.

What technologies are used?

Python for ML and backend, Flask as the web framework, scikit-learn for model training, Pandas and NumPy for data processing, and HTML/CSS for the frontend interface.

What features does PhishNet extract from a URL?

PhishNet extracts 20 structural features including URL length, number of dots and hyphens, presence of IP address, HTTPS usage, URL entropy (randomness), subdomain count, digit ratio, suspicious keywords like login/verify/bank, and more. No live network lookups are needed.

Why is there a "Proceed with Caution" option for unsafe URLs?

No ML model is 100% accurate — there is a small chance of False Positives where a legitimate URL is wrongly flagged. The Proceed with Caution option lets experienced users visit the URL after acknowledging the risk, while still clearly warning others.

What dataset was used to train PhishNet?

PhishNet was trained on the Malicious URLs dataset containing 52,000+ labelled URLs across four categories: benign, phishing, malware, and defacement. SMOTE oversampling was applied to balance the classes before training.

What is the base paper?

Kasar, A. et al. (2025). Phishing URL Detection using Machine Learning Algorithms. IEEE ICAMIDA Conference. DOI: 10.1109/ICAMIDA64673.2025.11208974
